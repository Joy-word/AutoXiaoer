Capabilities

为什么是「小二」

像人一样通过界面理解与操作界面；主动计划未来任务；有自我迭代能力的成长型智能体。

📱

纯端侧运行

App 独立完成规划与触控，无需通过电脑或 ADB 连线，出门在外同样可用。

🎭

百变小二

基于双 Agent + 人设 LLM 架构，自由定义人设，把「执行任务」与「对人说话」分得清清楚楚。

👁️

视觉理解，真机操作

看图识屏后再点击、滑动、输入，适配你手机里已安装的各类应用。

⏰

定时与通知唤醒

按日历重复执行预设任务；也可在指定 App 推送到达时自动开跑流水线。

💬

微信远程（ClawBot）

扫码绑定后可从微信投递指令，也是你与小二的专属连接通道。

🔌

OpenAI 兼容端点

不同模块的模型可分别配置；支持多配置切换与自定义 Prompt。

How it works

三步上手

从装好依赖到发出第一条自然语言任务，只需几分钟。详细图文见仓库 README。

安装并启动 Shizuku

通过无线调试等方式授权，让本机拥有执行点击、滑动等操作的能力。

安装 Auto 小二并授权

从 Releases 安装 APK，按引导打开悬浮窗、Shizuku、输入法等必要权限。

配置模型，开始说话

填入兼容 OpenAI 格式的 API；为「控制者」与「执行者」选好模型后即可下任务。
